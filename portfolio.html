Bookstore WebAPI

Este é um projeto é uma Web API desenvolvida usando o framework .NET, disponível no GitHub. O projeto inclui as seguintes funcionalidades:

  • Entidades: Duas entidades principais, Livros e Gêneros, onde um gênero pode ter vários livros.
  • CRUD Completo: Implementação de endpoints para criar, ler, atualizar e excluir dados.
  • Arquitetura: Uso de repositórios e serviços para organizar o acesso aos dados e a lógica de negócios.
  • DTOs: Uso de Data Transfer Objects (DTOs) para estruturar as respostas das APIs.
  • Middleware de Exceção Global: Tratamento centralizado de exceções para uma melhor gestão de erros.
  • Autenticação JWT: Implementação de autenticação segura usando JSON Web Tokens (JWT).
  • Identity e Criação de Tabelas: Uso do ASP.NET Identity para a criação e gestão de tabelas relacionadas à autenticação.
  • Rate Limiting: Limitação de taxa de requisições para proteger a API contra abusos.
  • Fluent API: Uso da Fluent API para configurar o mapeamento das propriedades das entidades e as regras de relacionamento.
  • MySQL: Uso do Pomelo para persistência de dados no banco de dados MySQL.
